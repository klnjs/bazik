Spinner

An element for illustrating ongoing work / progress.

Showcase​

  • index.tsx
  • index.module.css
import { Spinner, SpinnerTrack, SpinnerThumb } from '@klnjs/spinner'
import classes from './spinner.module.css'

export default () => (
  <Spinner className={classes.spinner}>
    <SpinnerTrack className={classes.track} />
    <SpinnerThumb className={classes.thumb} />
  </Spinner>
)
.spinner {
  display: block;
}

.thumb {
  color: var(--ifm-color-primary);
}

.track {
  color: var(--ifm-background-surface-color);
}

Features​

  • Configurable dimensions and duration.

  • Custom easing function.

Installation​

bun install @klnjs/spinner

Reference​

Spinner (Root)​

Contains all the parts of an spinner.

PropTypeDefault
asChildbooleanfalse
sizenumber24
widthnumber4

SpinnerTrack​

Used to optionally render a track (circle). It will automatically calculate its dimensions based on the size and width property of the root.

PropTypeDefault
asChildbooleanfalse

SpinnerThumb​

Used to render a thumb (circle section). It will automatically calculate its dimensions based on the size and width property of the root. It is possible to render multiple thumbs, and have them perform different animations based on the properties given.

PropTypeDefault
asChildbooleanfalse
anglenumber0
arcnumber25
capenumround
durationnumber1
easingstring-
